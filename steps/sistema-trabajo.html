SISTEMA DE TRABAJO

Haro y Kleiner define el sistema de trabajo “como una interacción combinada, a cualquier nivel de complejidad, de las personas, materiales, herramientas, maquinas, software, instalaciones y procedimientos diseñados para trabajar juntos para algún propósito común, adicionalmente un sistema puede tener una combinación de pequeños subsistemas”

Fuente: HARO, Elizabeth y KLEINER, Brian M. Macroergonomics as an organizing process for systemssafety. En: Applied ergonomics. Vol. 39 (2008); p. 450-458.
